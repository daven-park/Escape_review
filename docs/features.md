# Features

## User Stories & Feature Spec

---

## 1. 인증 (Authentication)

### 1.1 Google OAuth 로그인
- Google 계정으로 1-click 로그인
- 신규 사용자는 자동 회원가입
- 프로필 이미지 Google 아바타 자동 적용

### 1.2 이메일/비밀번호 로그인
- 이메일 + 비밀번호로 회원가입 및 로그인
- 비밀번호 분실 시 이메일 재설정 링크 발송

### 1.3 JWT 세션 관리
- Access Token (1시간) + Refresh Token (30일)
- 자동 토큰 갱신
- 로그아웃 시 Refresh Token 무효화

---

## 2. 지역/매장 탐색 (Explore)

### 2.1 지역별 매장 목록
- 시/구 단위 지역 필터
- 매장 카드: 이름, 대표 이미지, 평점, 테마 수
- 무한 스크롤 (cursor-based pagination)

### 2.2 매장 상세
- 매장 정보: 이름, 주소, 전화번호, 웹사이트
- 소속 테마 목록 (카드 그리드)
- 지도 링크 (카카오맵 / 네이버지도)

### 2.3 테마 상세
- 기본 정보: 이름, 장르, 난이도(1-5), 최소/최대 인원, 소요 시간
- 공포 지수 (0-5)
- 예약 링크 (외부 사이트)
- 후기 목록
- 평균 평점 / 별점 분포

---

## 3. 후기 (Reviews)

### 3.1 후기 작성
- 별점 (1-5, 0.5 단위)
- 텍스트 후기 (최소 20자, 최대 2000자)
- 이미지 첨부 (최대 5장, Cloudflare R2)
- 체감 난이도 선택 (쉬움/보통/어려움/매우어려움)
- 플레이 날짜 기록
- 스포일러 경고 토글

### 3.2 후기 목록
- 최신순 / 인기순 (좋아요) 정렬
- 스포일러 후기는 블러 처리 (클릭으로 공개)
- 이미지 갤러리 뷰

### 3.3 후기 좋아요
- 로그인한 사용자만 가능
- 좋아요 수 실시간 표시

### 3.4 후기 수정/삭제
- 작성자 본인만 가능
- 삭제 시 soft delete (admin 복구 가능)

---

## 4. 북마크 (Bookmarks)

### 4.1 테마 북마크
- 하트 아이콘으로 토글
- 북마크한 테마 목록 → 프로필 페이지

### 4.2 북마크 관리
- 북마크 목록: 테마명, 매장명, 장르, 난이도
- 북마크 취소 (목록에서 직접)

---

## 5. 예약 슬롯 확인 (Availability)

### 5.1 날짜별 슬롯 조회
- 달력 UI에서 날짜 선택
- 해당 날짜의 시간대별 잔여 여부 표시
- 예약 가능한 슬롯 클릭 → 외부 예약 사이트 이동

### 5.2 슬롯 데이터
- 관리자가 슬롯 등록 (또는 외부 API 연동 예정)
- Redis 캐시로 잔여 슬롯 빠른 조회 (30초 TTL)

---

## 6. 검색 (Search)

### 6.1 통합 검색
- 테마명, 매장명, 지역명 통합 검색
- PostgreSQL Full-Text Search (한국어 지원)
- 검색 결과: 테마 / 매장 탭 분리

### 6.2 필터
- 장르 (복수 선택 가능)
- 난이도 범위
- 인원수
- 지역
- 평점 이상

---

## 7. 사용자 프로필 (Profile)

### 7.1 내 프로필
- 닉네임, 아바타 수정
- 작성한 후기 목록
- 북마크한 테마 목록
- 통계: 총 플레이 수, 평균 평점

### 7.2 타 사용자 프로필 (공개)
- 작성한 후기만 공개
- 북마크 목록은 비공개

---

## 8. 관리자 (Admin) — Phase 8

### 8.1 매장/테마 관리
- 매장 등록 / 수정 / 삭제
- 테마 등록 / 수정 / 삭제

### 8.2 슬롯 관리
- 날짜/시간별 슬롯 등록
- 슬롯 가용 여부 수동 토글

### 8.3 후기 관리
- 신고된 후기 검토 및 삭제

---

|---|---|

| 다국어 | 한국어 (초기), 영어 (추후) |
